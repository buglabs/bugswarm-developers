API Key Management Tool

A user's API keys are heavily relied upon for authorization when using the BUGswarm API. Use this tool to create new API keys and reference already existing API keys. Creating new API keys will simply overrite old API keys if they exist. Note that this tool uses the methods laid out in the API Keys section of the Configuration API.
