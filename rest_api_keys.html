API Keys

Manage your API keys with the REST API.

Create

  • URL: http://api.bugswarm.net/keys
  • Method: POST
  • Authentication: "username:password"
  • Parameters (Required): None
  • Parameters (Optional): Key type

Each user must possess at least a master API key. In addition to the master key, producer and consumer keys can be generated. This method is used to create API keys. If a key already exists, this method will replace that key with a new one. Users have the option of specifying the type of key they wish to create in the URL. For instance, http://api.bugswarm.net/keys/master will generate a master key, while http://api.bugswarm.net/keys/producer will generate a producer key. By default, if no key type is specified, a master key is created. Each key holds a different set of permissions with regards to participating in swarms.

Usage: 
curl -X POST --user "username:password" http://api.bugswarm.net/keys
curl -X POST --user "username:password" http://api.bugswarm.net/keys/master
curl -X POST --user "username:password" http://api.bugswarm.net/keys/producer
curl -X POST --user "username:password" http://api.bugswarm.net/keys/consumer
Returns: The created key in JSON format. 
{
	"apikey": "911cbc244827dba9c6dbdd09aff3e5e8fe5eee02",
  	"type": "master" 
}

List

  • URL: http://api.bugswarm.net/keys
  • Method: GET
  • Authentication: "username:password"
  • Parameters (Required): None
  • Parameters (Optional): Key type

A user's API key (generally the master key) must be passed as a header when using the rest of the methods in this API as a form of account authorization. This method will list the API keys associated with the user's account. Users may specify which key the would like to retrieve through the URL. For instance, http://api.bugswarm.net/keys/master will retrieve the master key, while http://api.bugswarm.net/keys/producer will retrieve the producer key, if one exists.

Usage: 
curl -X GET --user "username:password" http://api.bugswarm.net/keys
curl -X GET --user "username:password" http://api.bugswarm.net/keys/master
curl -X GET --user "username:password" http://api.bugswarm.net/keys/producer
curl -X GET --user "username:password" http://api.bugswarm.net/keys/consumer
Returns: The desired key(s) and their relative information in JSON format.
