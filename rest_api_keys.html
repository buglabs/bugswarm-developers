API Keys

Manage your API keys with the REST API.

Create

  • URL: http://api.bugswarm.net/keys
  • Method: POST
  • Authentication: "username:password"
  • Parameters (Required): None
  • Parameters (Optional): type

Each user must possess at least a master API key. In addition to the master key, producer and consumer keys can be generated. This method is used to create API keys. If a key already exists, this method will replace that key with a new one. Users have the option of specifying the type of key they wish to create in the URL. For instance, http://api.bugswarm.net/keys/master will generate a master key, while http://api.bugswarm.net/keys/producer will generate a producer key. By default, if no key type is specified, a master key is created. Each key holds a different set of permissions with regards to participating in swarms.

Returns: The created key in JSON format.
